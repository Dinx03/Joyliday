# 🎯 Joyliday - Employee Management System

ระบบจัดการพนักงานและสมาชิก 🎉  
ใช้ **Flask + MySQL** รองรับ CRUD Operations  

## 🛠 Tech Stack
- **Backend:** Flask (Python)
- **Database:** MySQL
- **Frontend:** HTML, CSS, Bootstrap
- **Tools:** VS Code, Anaconda, GitHub
